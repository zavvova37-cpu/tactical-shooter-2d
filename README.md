# codex-code-generator

Локальный 2D тактический шутер для двух команд (Синие против Красных), рассчитанный на игру на одном устройстве (ПК или мобильном).

## Что реализовано

- Выбор персонажа для каждой команды перед началом матча.
- Локальный матч в ограниченной арене с укрытиями.
- Виртуальные джойстики и кнопки (огонь/граната/смена оружия) для мобильных устройств.
- Клавиатурное управление для ПК.
- Несколько типов оружия: пистолет и автомат.
- Гранаты с простой физикой: полёт, гравитация, отскоки от стен/укрытий и взрыв по таймеру.
- Подсчёт очков между раундами.

## Запуск

1. Откройте `index.html` в браузере.
2. Выберите персонажей команд.
3. Нажмите **«Начать матч»**.

## Управление

### Игрок A (Синие, ПК)
- Движение: `W A S D`
- Огонь: `F`
- Граната: `G`
- Смена оружия: `Q`

### Игрок B (Красные, ПК)
- Движение: `↑ ← ↓ →`
- Огонь: `K`
- Граната: `L`
- Смена оружия: `P`

На мобильных устройствах используйте экранные джойстики и кнопки.
